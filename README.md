# 📅 Kalender – Feiertage für Augsburg / Bayern (iCal .ics)

Dieses Repository enthält eine oder mehrere **iCalendar (.ics)** Dateien zur Nutzung in Kalenderanwendungen wie Apple Kalender, Google Kalender oder Outlook.

Die Daten können als **Kalenderabo** eingebunden werden und aktualisieren sich automatisch bei Änderungen im Repository.

---

## 🚀 Nutzung (Kalender abonnieren)

### Apple Kalender (macOS / iOS)

1. Apple Kalender öffnen  
2. **Ablage → Neues Kalenderabonnement**  
3. folgende URL einfügen: https://raw.githubusercontent.com/lngsamu/augsburg-holidays/refs/heads/main/augsburg_holidays.ics
4. Aktualisierungsintervall auswählen (z. B. stündlich oder täglich)

---

### Google Kalender

1. Google Kalender öffnen  
2. „Weitere Kalender“ → **Per URL hinzufügen**  
3. gleiche URL einfügen wie oben  
4. bestätigen

---

## 📂 Inhalt

Die `.ics` Datei enthält u. a.:

- Feiertage (z. B. regionale Feiertage)
- ggf. spezielle Ereignisse oder Kalenderdaten
- konsolidierte Kalenderdaten für mehrere Jahre

---

## 🔄 Updates

Der Kalender aktualisiert sich automatisch, sobald:

- die `.ics` Datei im Repository geändert wird
- der Kalender in der App synchronisiert wird

---

## ⚠️ Hinweise

- Dies ist ein **öffentliches Kalender-Abo**
- Änderungen werden automatisch übernommen
- Kein Anspruch auf Vollständigkeit oder Korrektheit der enthaltenen Daten.
- Dieser Kalender ist unabhängig und wird nicht offiziell von Apple, Google oder anderen Anbietern bereitgestellt.

---

## 🛠 Technische Details

- Format: iCalendar (.ics)
- Encoding: UTF-8
- kompatibel mit allen gängigen Kalender-Apps
